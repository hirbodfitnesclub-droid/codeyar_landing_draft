## 📝 جزئیات فنی

### تغییرات متن (Localization)
- متن های داخل چک‌لیست از فارسی → انگلیسی
- نمادهای فارسی را با معادل انگلیسی جایگزین کنید
- هیچ‌کدام از متن‌های منطق‌ی (State Names) نباید تغییر کند

### تغییرات Layout (Direction)
- **تمام Flex Items:** از `flex-row` استفاده کنید (نه `flex-row-reverse`)
- **Padding و Margin:** توجه به اینکه `pl-` (padding-left) باید متناسب با چپ‌چین‌بودن تنظیم شود
- **Text Alignment:** تمام متن‌ها باید `text-left` باشند
- **Icon Positioning:** آیکون‌ها باید سمت چپ متن قرار گیرند (نه سمت راست)

### فایل‌های مرتبط
```
CONTEXT_FILES: [
  "components/ui/task-progress-card.tsx",
  "app/globals.css"
]
```

### نکات مهم
- ⚠️ فقط UI متن‌ها را تغییر دهید، منطق React را لمس نکنید
- ⚠️ تمام Spacing و Layout باید Responsive باقی بماند
- ⚠️ Tailwind Classes را برای Direction تغییر استفاده کنید
