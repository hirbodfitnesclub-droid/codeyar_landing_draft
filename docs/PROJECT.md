# PROJECT.md

## هدف بیزینس
بهینه‌سازی و بومی‌سازی لندینگ‌پیج «کدیار» جهت تبدیل به یک قیف فروش (Sales Funnel) متمرکز بر ارزش پیشنهادی هسته‌ای: محیط توسعه مبتنی بر هوش مصنوعی با «حافظه پروژه» برای جلوگیری از توهم AI، هدف‌گیری دقیق وایب‌کدرها و توسعه‌دهندگان ایرانی جهت رفع چالش‌های فیلترینگ و هزینه‌های ارزی.

## پرسونای هدف
وایب‌کدرهای نوپا، فریلنسرهای خسته از تحریم و قطعی کانتکست در ابزارهای خارجی، و بنیان‌گذاران استارتاپ‌ها که به دنبال تسریع فرآیند ساخت MVP با حداقل هزینه و بدون نیاز به دانش عمیق مهندسی نرم‌افزار هستند.

## پشته تکنولوژی (Tech Stack)
- **فریم‌ورک:** Next.js (App Router)
- **زبان:** TypeScript
- **استایل‌دهی:** Tailwind CSS
- **کامپوننت‌ها:** React Functional Components

## [حیاتی] نبایدهای سخت‌گیرانه (Anti-Patterns)
- **استایل‌دهی خطی (Inline CSS) مطلقاً ممنوع است:** تمامی تغییرات ظاهری باید صرفاً از طریق کلاس‌های Tailwind CSS در پراپ `className` اعمال شوند.
- **تغییر منطق کامپوننت ممنوع:** به هیچ وجه State (مثل `useState`) یا هوک‌های اضافی به کامپوننت‌های UI/Sections اضافه نشود، مگر اینکه برای اعمال دیتای جدید (مثل آرایه‌ها) ضروری باشد.
- **افزودن پکیج جدید ممنوع:** تغییرات فعلی صرفاً در سطح کپی‌رایتینگ و دیتای استاتیک است. استفاده از کتابخانه‌های انیمیشن جدید یا مدیریت استیت مجاز نیست.
- **متن جایگزین (Lorem Ipsum) ممنوع:** فقط و فقط از متون دقیق مارکتینگی ارائه شده در تسک‌ها استفاده شود.
