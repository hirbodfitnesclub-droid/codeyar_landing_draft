# PROJECT.md — ستاره قطبی پروژه کدیار

## هدف بیزینس
کدیار یک لندینگ پیج فارسی است که محیط توسعه مبتنی بر هوش مصنوعی با حافظه طولانی‌مدت را معرفی می‌کند. هدف: جذب وایب‌کدرهای ایرانی و تبدیل آن‌ها به کاربر پولی.

## پرسونای هدف
- برنامه‌نویسان و علاقه‌مندان به کدنویسی با هوش مصنوعی (Vibe Coders)
- افرادی که تجربه ناموفق با ابزارهای AI Coding داشته‌اند (لوپ ارور، فراموشی کانتکست)
- محدوده سنی ۱۸-۴۵، فارسی‌زبان

## پشته تکنولوژی (Tech Stack)
| لایه | تکنولوژی |
|------|----------|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| Animation | Framer Motion |

## نبایدهای سخت‌گیرانه (Anti-Patterns)

### ممنوعیت‌های مطلق:
1. **CSS خطی (inline styles)** — فقط Tailwind CSS مجاز است
2. **استفاده از `!important`** — ترجیحاً از specificity صحیح استفاده شود
3. **Magic Numbers** — همه مقادیر باید از سیستم طراحی Tailwind باشد (نه `px` دستی)
4. **ساختار غیر-ماژولار** — هر کامپوننت باید در فایل جداگانه باشد
5. **انیمیشن‌های CSS خام** — فقط Framer Motion برای انیمیشن‌های پیچیده
6. **هاردکد کردن متون** — متون باید قابل تغییر باشند (ترجیحاً در بالای فایل یا constant)
7. **استفاده از `any` در TypeScript** — تایپ‌ها باید صریح باشند
8. **نادیده گرفتن Responsive** — هر تغییر UI باید mobile-first باشد

### الزامات طراحی:
- پالت رنگ: zinc (خنثی) + purple/violet (برند) + green (موفقیت) + blue (اکسنت)
- تم: Dark mode only
- فونت فارسی: Vazirmatn
- فونت کد: Geist Mono
- انیمیشن‌ها: ظریف، حرفه‌ای، بدون اغراق

### الزامات Responsive:
- **Mobile-first**: طراحی ابتدا برای موبایل، سپس scale-up
- **Breakpoints**: `sm` (640px), `md` (768px), `lg` (1024px), `xl` (1280px)
- **هیچ overflow افقی**: هیچ المانی نباید از عرض صفحه بیرون بزند
- **Touch-friendly**: دکمه‌ها حداقل 44px ارتفاع در موبایل
