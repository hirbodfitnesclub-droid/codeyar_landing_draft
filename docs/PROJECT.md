# PROJECT.md

## هدف بیزینس (Business Goal)
ایجاد یک پلتفرم توسعه و آموزش مدرن (کدیار) با ارائه یک تجربه بصری و تعاملی خیره‌کننده ("The Sentient Terminal") برای تبدیل شدن به انتخاب اول برنامه‌نویسان و بنیان‌گذاران.

## پرسونای هدف (Target Persona)
توسعه‌دهندگان نسل زد (Gen Z)، Vibe Coderها، و بنیان‌گذاران استارتاپ که دارای ذهنیت رشد، عمل‌گرا و به دنبال ابزارهای پیشگام و آینده‌نگرانه هستند.

## پشته تکنولوژی (Tech Stack)
- **فریم‌ورک اصلی:** Next.js (App Router)
- **زبان:** TypeScript
- **استایل‌دهی:** Tailwind CSS
- **انیمیشن و تعامل:** Framer Motion
- **آیکون‌ها:** Lucide React

## نبایدهای سخت‌گیرانه (Anti-Patterns)
- **ممنوعیت کامپوننت‌های مونولیتیک (Monolithic Components):** قرار دادن لاجیک‌های مختلف انیمیشن (Mouse Tracking, Scroll, Text Reveal) در یک فایل یکپارچه مطلقاً ممنوع است.
- **ممنوعیت State برای انیمیشن:** استفاده از `useState` برای مانیتور کردن موقعیت ماوس یا اسکرول اکیداً ممنوع است (باعث Re-render مخرب می‌شود). فقط و فقط از `useMotionValue` و `useTransform` استفاده شود.
- **ممنوعیت CSS خطی (Inline Styles):** به جز مواردی که Framer Motion برای مقادیر داینامیک نیاز دارد، هرگونه استایل‌دهی خطی ممنوع است. فقط Tailwind.
- **ممنوعیت فریم‌ورک‌های متفرقه انیمیشن:** استفاده از GSAP یا CSS Animations کلاسیک برای تعاملات پیچیده ممنوع است تا تداخل با Framer Motion ایجاد نشود.
